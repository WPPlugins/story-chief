This plugin lets you publish your articles straight from Story Chief to your Wordpress website.

== Description ==

Story Chief is a brand new tool that lets you push your content to different publishing channels.
It has a universal editor that lets you create beautiful stories with ease, making them look as they should on a range of different channels and measure your impact in one place.

With this plugin you can connect your wordpress website to Story Chief.
After the simple setup is completed you will be able to publish your articles from Story Chief straight to you Wordpress blog website.

Read statistics will also be available from Story Chief.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> Story Chief screen to configure the plugin
4. Copy over your encryption key from Story Chief
